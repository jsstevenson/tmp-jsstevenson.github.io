CSE 351: The Hardware-Software Interface

The classic undergrad computer systems course. Memory and data, floating-point numbers, basic assembly, memory (physical, virtual), caching.

x86-64 assembly, C
BIOST 310: Introduction to Biostatistics

Introduction to basic biostatistical concepts (study types, significance tests). Learned a bit more about medical academia than I expected.

CSE 374: Intermediate Programming Concepts

A grab bag of Bash/Linux and C fundamentals.

Not highly conceptual, but ultimately provided a great excuse to read through K&R.

Bash, C, C++
CSE 413: Programming Languages & Implementation

Coverage of introductory programming language theory (with case studies in Racket for functional programming and Ruby for OOP).

Additional discussion of basics of programming language implementation, culminating in the development of an interpreter for a fictional arithmetic language. This morphed into a project to implement a Python interpreter.

Racket, Ruby
AMATH 301: Introduction to Scientific Computing

A survey of a number of introductory problems and methods in scientific computing, including numerical approaches to differentiation and integration, least-squares regression, and more linear algebra, plus a general introduction to MATLAB.

MATLAB
CSE 373: Data Structures & Algorithms

Basic data structures (linked lists, binary search trees, AVL trees, hash tables), algorithms (emphasis on graph problems), and algorithm analysis (particularly of dynamic programming).

Course labs were in Java, but I started a repo of a Python implementation of the utilized data structures as practice.

Java
AMATH 352: Applied Linear Algebra & Numerical Analysis

Covered introductory matrix operations (inversion, decompositions, etc) from a largely computational perspective, with an eye to numerical stability and big-O growth.

I'm still a little unsettled by the fact that eigenvalues... work.

MATLAB
